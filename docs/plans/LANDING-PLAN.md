## 风险应对预案

### 高风险：首次运行失败
**预案**:
1. 检查日志定位问题
2. 参考 PRODUCTION-IMPROVEMENTS.md 错误分类
3. 在 GitHub 搜索类似问题
4. 降级使用（如先跳过 LLM，使用规则版 Agent）

### 中风险：VRAM 不足
**预案**:
1. 降低分辨率到 640x432
2. 减少帧数到 17
3. 使用 FP8 量化模型
4. 关闭其他 GPU 程序

### 中风险：生成质量不佳
**预案**:
1. 手动调优 Prompt
2. 尝试不同 seed
3. 调整采样步数和 CFG
4. 换不同的工作流
