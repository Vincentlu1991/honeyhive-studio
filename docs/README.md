# Project Documentation

## Folders

- `comfyui/` - ComfyUI setup notes, workflow guidance, and GPU presets.
- `commercial/` - Commercialization roadmap, deployment strategy, ownership map, and ChatGPT App plan.
- `guides/` - How-to documents and operational commands.
- `plans/` - Product, media, and project planning notes.
- `reports/` - Status reports, summaries, checklists, and completion notes.
- `research/` - Model and ecosystem research.
- `security/` - Threat model and security best-practices report.

The root-level `workflow_*.json` files are intentionally left in place because the current app configuration points to them from `agent-platform/.env`.
